## 📖 Overview
This repository is a collection of **Java CLI applications** built to practice and demonstrate programming concepts.  
Each file is a self‑contained project showcasing topics such as **file handling, collections, threading, lambdas, and algorithms**.  
The repo serves as a **portfolio of mini projects** documenting my programming journey.

---

## 🛠️ Technologies Used
- **Java** → Core language for all applications  
- **CLI (Command Line Interface)** → Text‑based interaction for apps  
- **File I/O** → Reading, writing, and managing files  
- **Collections Framework** → HashMap, HashSet, TreeMap, TreeSet, LinkedList  
- **Concurrency** → Threads, synchronization, producer‑consumer patterns  
- **Functional Programming** → Lambdas for cleaner, shorter code  
- **Algorithms** → Palindrome checkers, prime number validation, string utilities  

---

## 📂 Project Highlights
- **ExpenseTracker.java** → Track expenses with file persistence and validation  
- **LibrarySystemCLI.java** → Manage books and members using OOP principles  
- **ContactOrganizer.java** → Store and search contacts in a CLI interface  
- **QuizGame.java** → Interactive quiz showcasing arrays  
- **ProducerConsumerCLI.java** → Demonstrates threading and concurrency  
- **LambdaDemoCLI.java / LambdaComparisonCLI.java** → Cleaner code with lambdas  
- **WordFrequencyCounter.java** → Counts word occurrences in text  

*(There are many more files — each demonstrates a unique concept.)*
